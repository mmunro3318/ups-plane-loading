# UPS LoadBalancer (Plane Loading)

React + Vite app that simulates loading ULDs onto a Boeing 757 and optimizes the load order to improve balance (MAC%).

## Background / experiment
This is a continuation of a school project where I explored an MCTS-based approach to load ordering. For this iteration, I experimented with a simulated annealing optimizer and built a small, visual frontend to make it easier (and more fun) to compare strategies and see the results.

## Run locally
```bash
npm install
npm run dev
```

## Build / preview
```bash
npm run build
npm run preview
```
